# 🌋 Entre no Coração da Islândia — Serviços de Viagens

> Landing page premium e multipáginas desenvolvida para agência de turismo especializada em expedições exclusivas na Islândia. O projeto foca na apresentação de ambientes naturais deslumbrantes, lazer, observação da fauna local, expedições automotivas e eventos organizados em grupo.

## 🎯 Diferenciais e Funcionalidades do Projeto

*   **Arquitetura Multipáginas:** Estrutura robusta composta por **7 páginas completas** e totalmente responsivas (Início, Expedições detalhadas, FAQ, Contato, entre outras).
*   **Integração em Tempo Real (API):** Formulários de contato dinâmicos implementados em todas as 7 páginas, utilizando a **API do Formspree** para capturar e disparar os dados de leads instantaneamente para o Gmail.
*   **Foco em Conversão (UX/UI):** Pontos de contato estratégicos distribuídos em todo o layout, incluindo CTAs diretas para canais de atendimento e redes profissionais.
*   **Desenvolvimento Limpo (Vanilla Stack):** Projeto construído inteiramente com tecnologias puras, sem frameworks pesados, garantindo uma excelente performance de carregamento, SEO otimizado e total controle do código.
*   **Conformidade de Privacidade:** Banner de cookies integrado nativamente para consentimento e respeito às diretrizes de proteção de dados.

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Função no Projeto |
| :--- | :--- |
| **HTML5 Puro** | Estruturação semântica de todas as 7 páginas focada em acessibilidade e SEO. |
| **CSS3 Puro** | Estilização avançada, layout responsivo (Mobile-First) e paleta de cores premium com tema escuro. |
| **JavaScript Vanilla** | Manipulação de DOM, lógica de validação dos formulários e controle do banner de privacidade. |
| **Formspree API** | Gerenciamento de requisições HTTP e envio assíncrono dos dados de contato. |

## ⚙️ Como Executar o Projeto Localmente

1. Clone este repositório:
```bash
git clone https://github.com/ThiagoLLimas/servicos-viagens.git
```
Entre na pasta do projeto:

```Bash
cd servicos-viagens
```
Execução:
Como o projeto foi desenvolvido em Vanilla Stack (HTML, CSS e JS puros), você não precisa instalar nenhuma dependência!

Basta abrir o arquivo index.html diretamente no seu navegador, ou:

Utilizar a extensão Live Server no seu editor de código para rodar o projeto com recarregamento automático.
